# CoolEq 导星系统设计规范 (Guide Scope Design Specification)

本文档详细阐述了 CoolEq 全自动赤道仪系统中导星镜的物理结构设计理念、选型标准及软件校准策略。与传统“外挂式”导星镜不同，CoolEq 将其视为核心视觉传感器，强调绝对刚性与算法补偿。

## 1. 设计理念 (Design Philosophy)

*   **从“调节”到“刚性”**: 摒弃传统的三点式可调导星环，采用一体化刚性连接，消除差异形变 (Differential Flexure)。
*   **从“硬件同轴”到“软件补偿”**: 不追求物理上的完美同轴，利用软件计算 Offset 来修正指向偏差。
*   **视觉中枢**: 导星镜不仅用于导星，更是全自动极轴校准 (Auto PA) 和板解析 (Plate Solving) 的核心传感器。

## 2. 物理结构规范 (Mechanical Specs)

### 2.1 光学参数选型
| 参数 | 推荐范围 | 理由 |
| :--- | :--- | :--- |
| **焦距** | 120mm - 200mm | 兼顾视场大小（便于寻找校准星）与解析精度（保证极轴计算准确）。 |
| **口径** | 30mm - 50mm | 保证进光量，将校准曝光时间控制在 0.5s - 1s 级别。 |
| **接口** | CS / C Mount | 相比 1.25寸桶插接，螺纹连接更紧固，无松动风险。 |

### 2.2 机械连接结构
**禁止使用三点式导星环。** 推荐以下集成方案：

1.  **刚性抱箍 (Rigid Clamp)**: 使用 CNC 加工的一体式铝合金抱箍，将镜筒完全锁死在鸠尾板或主镜筒上。
2.  **把手集成 (Handle Integrated)**: 将光路设计在提手内部（类似 RedCat 51 把手），既节省空间又提供装甲级保护。
3.  **侧边直连 (Direct Bolt-on)**: 利用主镜筒侧面的 CNC 螺孔直接固定，减少力臂长度。

### 2.3 对焦机构
*   **一次性锁死**: 采用旋转物镜筒对焦，配合**锁紧环 (Lock Ring)** 死死顶住。
*   **防震处理**: 出厂前在对焦螺纹处点螺纹胶。
*   **光学策略**: 允许轻微离焦（星点稍胖），有助于质心计算，抑制热噪点干扰。

## 3. 调校策略 (Calibration Strategy)

### 3.1 为什么不需要物理同轴？
*   **板解析**: 只要能拍到星空，ASTAP 就能解算出当前导星镜的真实坐标。
*   **极轴校准**: 依赖的是坐标的**相对变化量**（旋转中心计算），对绝对指向不敏感。
*   **导星**: PHD2 仅需锁定画面内任意星点，不要求该星点位于主镜中心。

### 3.2 软件同轴校准 (Software Co-alignment)
针对 GOTO 指向的一致性，采用软件补偿方案：

1.  **物理安装**: 将导星镜刚性固定，肉眼粗略对准主镜方向（误差 5度以内皆可）。
2.  **标定流程**:
    *   主镜对准远处地标（如塔尖）至画面中心。
    *   导星镜拍照，识别塔尖在画面中的像素坐标 `(x, y)`。
    *   计算该坐标相对于画面中心 `(cx, cy)` 的偏移量 `Offset Vector`。
3.  **运行时补偿**:
    *   CoolEq Daemon 在执行 GOTO 解析同步时，自动应用此 Offset。
    *   *公式*: `主镜真实坐标 = 导星镜解析坐标 + Offset`。

## 4. 总结

| 特性 | 传统方案 | CoolEq 方案 |
| :--- | :--- | :--- |
| **连接方式** | 三点调节环 (易松动/形变) | 刚性抱箍/集成 (零形变) |
| **同轴调节** | 手拧螺丝 (繁琐/不稳定) | 软件参数补偿 (一次标定，永久有效) |
| **对焦** | 经常需要微调 | 出厂锁死 |
| **角色** | 辅助配件 | 核心视觉传感器 |
